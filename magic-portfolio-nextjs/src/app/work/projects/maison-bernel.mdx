## Resumen del Proyecto

Maison Bernel LU representa la convergencia entre el lujo y la tecnología. Se desarrolló una plataforma que no solo vende productos, sino que cuenta una historia de marca a través de interacciones fluidas.

## Características Principales

- **Narrativa Visual**: Uso de animaciones sutiles para guiar al usuario a través de la marca.
- **Catálogo Premium**: Visualización de productos optimizada para destacar detalles de alta calidad.
- **Experiencia de Usuario (UX)**: Interfaz intuitiva enfocada en la conversión suave.

## Tecnologías Utilizadas

- **Frontend Avanzado**: Implementación de interacciones de alto nivel.
- **Optimización de Medios**: Gestión eficiente de imágenes de alta resolución.
