## Resumen del Proyecto

Este proyecto consistió en la creación de una plataforma digital para la gestión de flota y reservas de una empresa de alquiler de yates. La prioridad fue simplificar el proceso de reserva para el cliente final y automatizar la gestión de calendarios.

## Características Principales

- **Gestión Automatizada**: Sistema de reserva en tiempo real que evita conflictos de horario.
- **Interfaz Exclusiva**: Estética de lujo alineada con el mercado de altamar.
- **Panel de Control**: Herramientas internas para la administración eficiente de la flota y el personal.

## Tecnologías Utilizadas

- **Framework Moderno**: Estructura robusta para aplicaciones de alto tráfico.
- **Automatización de Backend**: Flujos de trabajo personalizados para el manejo de reservas.
